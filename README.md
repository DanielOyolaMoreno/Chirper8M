**Descripción del proyecto:**

Aplicación desarrollada con Laravel que reproduce un microblog tipo "Chirper": permite crear, editar y listar publicaciones breves ("chirps").

Implementa los modelos `User` y `Chirps`, controladores HTTP, políticas de autorización, migraciones y seeders con datos de ejemplo, y vistas Blade para la interfaz de usuario.


**Levantar el proyecto:**

Requisitos previos: tener instalado PHP y Composer.

Para levantar el proyecto hay que ejecutar:

```
composer run dev
```
